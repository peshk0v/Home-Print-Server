# [🖨️ Домашний принт-сервер на Raspberry Pi — полный гайд 🚀](https://peshk0v.github.io/Home-Print-Server/)

> **📖 Описание:**  
> Этот гайд поможет вам превратить старый USB-принтер в современный сетевой принт-сервер с помощью Raspberry Pi.  
> Всё, что потребуется — следовать инструкции шаг за шагом.  
> Главный плюс: **установка всего одной командой** благодаря готовому скрипту.

---

## 📦 Что вам понадобится?

| Компонент | Рекомендации |
|-----------|---------------|
| 🥧 Raspberry Pi | Любая модель с Wi-Fi (3B, 4B, Zero 2 W) |
| 💾 microSD-карта | 8–32 ГБ, класс 10 |
| 🔌 Блок питания | 5В / 2.5–3А (официальный или качественный) |
| 🖨️ USB-принтер | Любой старый, но рабочий |
| 💻 Компьютер | Для подготовки SD-карты (Windows/Mac/Linux) |

## 🚀 Шаг 1: Готовим загрузочную флешку с Raspberry Pi OS (Lite)

1. 📥 **Скачайте Raspberry Pi Imager** с официального сайта: [raspberrypi.com/software](https://www.raspberrypi.com/software/)

2. 🧩 **Установите Imager** и запустите его.

3. 📌 **Выберите устройство** → ваша модель Raspberry Pi (например, Raspberry Pi 4).

4. 💿 **Выберите ОС** → **Raspberry Pi OS (Lite, 64-bit)**.  
   *Если ваша модель не поддерживает 64 бита, возьмите 32-битную версию.*

5. 🎚️ **Нажмите на шестерёнку** (настройки ОС) — это важно для «безголовой» установки!

   - **Задать имя узла (hostname):** `printserver`
   - **Включить SSH** → «Использовать пароль для аутентификации»
   - **Задать имя пользователя и пароль:**  
     - Имя: `pi`  
     - Пароль: придумайте надёжный (запомните его!)
   - **Настроить беспроводную LAN:**  
     - SSID: точное имя вашей Wi-Fi сети  
     - Пароль: от вашей Wi-Fi

6. 💾 **Выберите накопитель** → ваша microSD-карта.

7. ✅ **Нажмите «Записать»** и подтвердите. Дождитесь окончания записи.

---

## 🔌 Шаг 2: Подключаем Raspberry Pi к принтеру и питанию

1. 🔌 Подключите USB-принтер к Raspberry Pi.
2. 💳 Вставьте microSD-карту в Raspberry Pi.
3. ⚡ Включите питание (подключите блок питания).

> ⏱️ Дайте Raspberry Pi 1–2 минуты на первую загрузку и подключение к Wi-Fi.

---

## 💻 Шаг 3: Подключаемся по SSH

📡 **Найдите IP-адрес вашего Raspberry Pi:**

- Посмотрите в веб-интерфейсе вашего роутера → список подключённых устройств. Ищите `printserver`.
- Или используйте программу `Angry IP Scanner` / `nmap`.

🔐 **Подключитесь через терминал (командную строку):**

```bash
ssh pi@printserver.local
```
*Если не работает — используйте прямой IP: `ssh pi@192.168.x.x`*

При первом подключении введите `yes` и нажмите Enter, затем введите пароль пользователя `pi` (он не отображается при вводе).

---

## 🧙 Шаг 4: Запуск универсального установочного скрипта

✨ **Одна команда — и всё готово!** ✨

Скопируйте и выполните в терминале SSH:

```bash
curl -sSL https://raw.githubusercontent.com/peshk0v/Home-Print-Server/refs/heads/main/installer.sh | sudo bash
```

### ⚙️ Что делает скрипт?

- 🔄 Обновляет систему  
- 📦 Устанавливает **CUPS** (сервер печати), **Samba** (для Windows), **Avahi** (автоматическое обнаружение)  
- 👤 Добавляет пользователя `pi` в группу `lpadmin`  
- 🌐 Настраивает удалённый доступ к CUPS  
- 🖥️ Настраивает Samba для общего доступа к принтеру  
- 🧩 Устанавливает популярные драйверы (Gutenprint, HPLIP)  
- 🔁 Перезапускает все службы  
- 📟 Выводит IP-адрес и инструкцию  

⏳ **Процесс займёт 5–15 минут.** Не закрывайте терминал и не выключайте питание!

---

## 🖱️ Шаг 5: Добавление принтера через веб-интерфейс CUPS

1. 🌐 **Откройте браузер** на любом устройстве в той же сети.
2. 🔗 **Перейдите по адресу:**  
   `http://[IP-АДРЕС_RASPBERRY_PI]:631`  
   (например, `http://192.168.1.100:631`)
3. 🛠️ Нажмите вкладку **«Administration»** → **«Add Printer»**.
4. 🔑 Введите логин `pi` и пароль, который вы задали.
5. 🖨️ **Выберите ваш принтер** из списка «Local Printers».
6. ✅ **Поставьте галочку** «Share This Printer» (открыть общий доступ).
7. 🧩 **Выберите драйвер** (CUPS предложит подходящий; если нет — попробуйте «Raw» или найдите драйвер для вашей модели).
8. 🎉 Нажмите **«Add Printer»**.

---

## 📡 Шаг 6: Подключаем компьютеры к принт-серверу

Ваш принтер теперь доступен в сети!

### 🪟 Windows
- Откройте **«Панель управления» → «Устройства и принтеры»** → **«Добавление принтера»**.
- Система должна найти принтер автоматически. Если нет — выберите **«Добавить сетевой, беспроводной или Bluetooth-принтер»**.

### 🍏 macOS / iPhone / iPad
- Принтер появится автоматически в списке печати (благодаря Bonjour / AirPrint).

### 🐧 Linux
- Принтер обычно находится автоматически через настройки печати (CUPS / Avahi).

### 📱 Android
- Установите приложение **«Mopria Print Service»** или **«PrintHand»**. Они поддерживают IPP-протокол.

### 🔧 Ручное добавление по IP (если автообнаружение не работает)
- Используйте адрес:  
  `ipp://[IP-АДРЕС_PI]:631/printers/[ИМЯ_ПРИНТЕРА]`

## 🧠 Как это работает? (коротко)

| Компонент | Роль |
|-----------|------|
| 🖨️ **CUPS** | Сервер печати — принимает задания, управляет очередью, отправляет на принтер |
| 🔗 **Samba** | Обеспечивает совместимость с Windows — принтер виден как сетевой |
| ✨ **Avahi** | Zero‑конфигурация — автоматическое обнаружение (Bonjour) |

## ❗ Возможные проблемы и решения

| Проблема | Решение |
|----------|---------|
| 🌐 Raspberry Pi не виден в сети | Проверьте правильность Wi-Fi данных в Imager. Подождите 2 минуты. Посмотрите в роутере IP. |
| 🔌 Ошибка SSH «Connection refused» | Убедитесь, что вы включили SSH в настройках Imager. Перезапишите карту. |
| 🖨️ Принтер не появляется в CUPS | Проверьте `lsusb` в SSH — должен быть список USB устройств. Если принтер есть, перезагрузите Pi: `sudo reboot`. |
| 📄 Задание в очереди, но не печатает | Проблема с драйвером. В CUPS попробуйте другой драйвер или отправьте тестовую страницу. |
